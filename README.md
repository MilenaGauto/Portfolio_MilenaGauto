# 💻 Mi Portfolio Profesional - Milena Gauto

Este repositorio contiene el código fuente de mi portfolio personal, desarrollado para mostrar mi trayectoria como estudiante de Ingeniería en Sistemas y mi trabajo como desarrolladora Backend.

**🌐 Link en vivo:** [https://milenagauto.github.io/Portfolio_MilenaGauto/](https://milenagauto.github.io/Portfolio_MilenaGauto/)

---

## 🚀 Tecnologías Utilizadas

Para el desarrollo de este sitio, utilicé un stack de frontend moderno centrado en la performance y la accesibilidad:

* **HTML5:** Estructura semántica del contenido.
* **CSS3:** Estilo personalizado, incluyendo el uso de Flexbox y CSS Grid para el diseño responsivo.
* **JavaScript:** Lógica de navegación y efectos visuales.
* **GitHub Pages:** Hosting gratuito y despliegue continuo.

---

## 🛠️ Características Principales

* **Diseño Responsivo:** Optimizado para dispositivos móviles, tablets y desktop.
* **Sección de Proyectos:** Galería interactiva que destaca mi sistema de gestión (Python/Django).
* **Dark Mode / Tech Aesthetic:** Estética inspirada en el desarrollo de software.
* **Optimización SEO:** Implementación de meta-tags para mejorar la visibilidad en buscadores.
